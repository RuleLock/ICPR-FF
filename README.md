# ICPR-FF

![py36][py36]

[Tesseract环境配置](https://github.com/RuleLock/ICPR-FF/blob/master/docs/Tesseract%20OCR.md)

运行前先安装依赖

`pip3 install -r requirements.txt`

将待分析的图片放于`images`路径下

运行`python3 main.py`

输出结果在`outputs`路径下

> 目前成功率 1/4，分析准确率：极低
